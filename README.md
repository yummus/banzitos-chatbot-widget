# Banzitos® Chatbot Widget

Widget de soporte al cliente para banzitos.com.mx.

## Instalación en Shopify

Pegar en `theme.liquid` antes de `</body>`:

```html
<script src="https://cdn.jsdelivr.net/gh/yummus/banzitos-chatbot-widget@main/widget.min.js"></script>
```

## Características
- Chat inteligente con base de conocimiento de productos
- Respuestas sobre envíos, alérgenos, puntos de venta
- Escalamiento a equipo de soporte
- Shadow DOM (no interfiere con estilos del sitio)
- Responsive y mobile-friendly
